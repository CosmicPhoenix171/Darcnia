Modules in this folder:

- engine.js: Canvas engine, camera, grid rendering (square/hex), input handling
- map.js: Tile grid model and serialization
- tokens.js: Token data and movement/selection logic
- fog.js: Fog of War overlay and reveal/hide methods
- generator.js: Simple procedural map/dungeon generator with traps/treasures/enemies
- network.js: Networking adapters (BroadcastChannel implemented, Firebase placeholder)
- ui.js: UI wiring, toolbar actions, initiative/chat, save/load, dice roller
